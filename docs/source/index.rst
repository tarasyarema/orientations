=======================================
Enumeration of k-connected orientations
=======================================

Prerequisites
-------------

This package are needed (you may already have them installed)::

   sage -pip install sage-package sphinx

Installation
------------

To use this module, you need to install from a terminal it via:: 

   sage -pip install orientations


Once installed you can use it directly from Sage::

   from orientations import *
